#  Resistor Code App

A Flutter-based **Resistor Color Code Calculator** designed to help electronics enthusiasts, students, and engineers quickly decode resistor values, calculate voltage dividers, and more.

---

## Features

 Identify resistor values from color bands  
Convert numerical values back to color codes  
Calculate voltage divider values  
Dark mode for comfortable use at night
